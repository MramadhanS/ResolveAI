# Syarat Ketentuan dan Kebijakan Privasi Data MyPertamina (Subsidi Tepat)

## 1. Keamanan dan Kewajiban Pengguna di Pangkalan (Syarat Pendaftaran)
**Kebijakan:** 
- Saat mendaftar secara tatap muka (misal: di Pangkalan LPG), pengguna **wajib menjaga keamanan data/dokumen pribadinya**.
- Pengguna **dilarang keras** menyerahkan dokumen KTP, Kartu Keluarga (KK), atau salinannya untuk disimpan atau ditahan oleh pengurus Pangkalan maupun pihak mana pun.
- Pengurus Pangkalan hanya bertugas menginput data melalui Web Subsidi Tepat resmi, dan KTP hanya untuk ditunjukkan sebagai bukti identitas saat pendaftaran atau transaksi.

## 2. Pengumpulan Data Pribadi
**Kebijakan:** Pertamina mengumpulkan Data Pribadi pengguna secara sukarela untuk keperluan pendaftaran, transaksi, dan kelancaran layanan.
**Data yang dikumpulkan secara langsung meliputi:**
- Nama lengkap, NIK (Nomor Induk Kependudukan), Nomor KK.
- Alamat email, nomor telepon, dan alamat sesuai KTP/lokasi usaha.
- Foto diri, Foto KTP, dan Foto Tempat Usaha (untuk kategori Usaha Mikro/Pengecer).
- Titik koordinat lokasi usaha.
- Data pembayaran saat melakukan transaksi.

**Data yang terekam otomatis oleh sistem meliputi:**
- Data lokasi riil atau perkiraan (Alamat IP, geo-location, lokasi Wi-Fi).
- Waktu aktivitas (pendaftaran, login, transaksi).
- Data perangkat (model *hardware*, sistem operasi, nomor IMEI, pengenal perangkat unik).
- Data catatan/log server.

## 3. Penggunaan dan Pembagian Data Pribadi
**Kebijakan:** Pertamina berkomitmen **tidak akan menjual, mengalihkan, atau mendistribusikan** Data Pribadi kepada pihak lain tanpa persetujuan pengguna, kecuali untuk tujuan berikut:
- **Verifikasi dan Subsidi:** Memproses data untuk keperluan penyaluran barang bersubsidi agar tepat sasaran.
- **Pencegahan Penipuan (Fraud):** Mencegah, mendeteksi, dan menyelidiki tindakan ilegal atau kecurangan. Pertamina berhak melakukan pemblokiran akun jika terbukti ada *fraud*.
- **Pihak Ketiga & Afiliasi:** Data dapat dibagikan secara terbatas kepada PT Pertamina (Persero), anak perusahaan, mitra usaha (seperti Pangkalan atau penyedia layanan pembayaran), dan vendor IT untuk menjalankan layanan Web Subsidi Tepat.
- **Hukum & Audit:** Data dapat diungkapkan kepada aparat penegak hukum atau instansi negara yang berwenang untuk mematuhi kewajiban hukum atau keperluan audit penyaluran subsidi.

## 4. Keamanan dan Penyimpanan Data
**Kebijakan:**
- Pertamina melindungi Data Pribadi dari akses tidak sah menggunakan tindakan dan prosedur keamanan, termasuk perlindungan lewat Kata Sandi/PIN pengguna.
- Meskipun upaya terbaik telah dilakukan, pengiriman data melalui internet tidak pernah 100% aman, sehingga pengguna menanggung risiko pengiriman informasi.
- Pertamina akan **menghapus dan/atau menganonimkan** Data Pribadi apabila data tersebut sudah tidak lagi diperlukan untuk tujuan pengumpulannya, atau sudah tidak diwajibkan oleh peraturan perundang-undangan.

## 5. Hak Pengguna (Perbaikan Data dan Penarikan Persetujuan)
**Kebijakan:**
- **Koreksi Data:** Pengguna berhak mengakses atau memperbaiki Data Pribadi yang salah melalui Layanan Pelanggan (Pertamina 135). Pengguna wajib menyertakan bukti identitas yang memadai untuk proses ini.
- **Penarikan Persetujuan:** Pengguna dapat menarik kembali persetujuan pemrosesan Data Pribadi melalui Pertamina 135.
- **Konsekuensi Penarikan:** Jika persetujuan ditarik, Pertamina tidak lagi memproses data tersebut. Konsekuensinya, hubungan hukum berakhir dan pengguna **tidak dapat lagi menggunakan layanan Web Subsidi Tepat**.

## 6. Layanan Pengaduan
**Kebijakan:** Jika pengguna merasa privasinya dilanggar atau memiliki keluhan terkait keamanan data, pengguna dapat menghubungi layanan pelanggan Pertamina di nomor telepon **135** atau melalui email **pcc135@pertamina.com**.
